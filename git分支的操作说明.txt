git分支操作
    查看分支
        查看分支
        git branch
        
        创建分支
        git branch 分支名 
        
        切换分支
        git checkout 分支名
        （git checkout -b //表示创建并切换分支）
        
        删除分支
        git branch -d 分支名
        
        合并分支
        git merge 被合并的分支名

        将本地分支提交到远端
        git push origin 分支名

        (测试增加行到dev分支)
